Steps to create [cell](https://www.google.com/search?q=mobile+cell+sector+coverage+base+station)
- Define location
- Type specific coordinates in:
- [Projected coordinate](https://www.google.com/search?q=projected+coordinate+system+[UTM](https://www.google.com/search?q=UTM+Universal+Transverse+Mercator+projection)+[GIS](https://www.google.com/search?q=GIS+Geographic+Information+System)) system – X and Y coordinates;
- Geographical [coordinate system](https://www.google.com/search?q=coordinate+reference+system+CRS+GIS) – Latitude and Longitude
coordinates
- Click on the map: coordinates will be filled
automatically from point defined on the map.
- Define direction ([azimuth](https://www.google.com/search?q=antenna+azimuth+direction+degrees+north)) – this will be
required if Cell object coordinates will be taken
by click on the map.
- Define name
- Choose Template or fill parameters manually
