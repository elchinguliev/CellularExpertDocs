# 3.1.14 Street view

Click this button ![icon](../../../assets/images/ce-express/user-guide-v73/p134-img1.png) to open [Street view](#kw:navigating-street-view:ce-express-street-view) tool.

This interface links a 2D map with a street-level view. The map shows spatial context and landmarks. The street view shows the real-world appearance of the same location. Both panels update together when you select a place.
