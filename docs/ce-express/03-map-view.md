# Map View

> **Version:** CE Express v7.2

The CE Express [Map view](#kw:layout:none) is the primary workspace for network planning and analysis.

## Layout

The [Map view](#kw:layout:none) is divided into six areas:

| Area | Description |
|------|-------------|
| 1. Top bar | Inventory3D window layout, settings, user menu |
| 2. [Left toolbar](#kw:left-toolbar-tools:none) | [Data management](#kw:31-data-management-tools:inventory3d-user-guide) and visualization tools |
| 3. Map canvas | Graphical [data management](#kw:31-data-management-tools:inventory3d-user-guide), object placement |
| 4. Table panel | Attribute data management (bottom) |
| 5. Results panel | Data analysis tools and calculation results |
| 6. Graphs panel | Calculation result graphs, additional analysis |

All main tools are accessed from the [left toolbar](#kw:left-toolbar-tools:none):

| Tool | Icon | Function |
|------|------|----------|

| Layers | 🗺 | Control map layer visibility |
| [Prediction History](#kw:44-step-4-prediction-history-and-result-management:ce-express-tr-rf) | 📊 | Browse and manage [prediction results](#kw:viewing-results:ce-express-rf-prediction) |
| Antennas | 📡 | [Antenna pattern](#kw:managing-the-antenna-library:ce-express-antenna) library |
| Geodata Sets | 🌍 | Manage terrain, [clutter](#kw:clutter-classification-values:ce-express-geodata), building data |
| Feature Templates | 📋 | Reusable object configurations |
| Prediction [Models](#kw:31-models:ce-express-tr-models) | 🧮 | Configure propagation [models](#kw:31-models:ce-express-tr-models) |
| Profile | 📈 | [Line-of-sight](#kw:running-a-profile:ce-express-profile) and [terrain profile](#kw:reading-the-profile-graph:ce-express-profile) |
| [Quick RF Prediction](#kw:quick-rf-prediction:ce-express-rf-prediction) | ⚡ | Rapid what-if prediction |
| RF Prediction | 📻 | Full coverage prediction |
| 3D RF Prediction | 🏙 | 3D propagation with buildings |
| Model Tuning | 🔧 | Calibrate models against measurements |

## View Modes

Switch between [view modes](#kw:view-modes:none) using the toolbar buttons:

- **[Map view](#kw:layout:none)** — standard 2D map display
- **Full screen table** — expand the attribute table to full screen
- **Split vertical** — map and table side by side
- **Split horizontal** — map above, table below
