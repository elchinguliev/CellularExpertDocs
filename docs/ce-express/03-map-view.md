# Map View

The [CE Express](https://www.google.com/search?q=Cellular+Expert+CE+Express+web+platform) Map view is the primary [workspace]([https](https://www.google.com/search?q=HTTPS+[SSL](https://www.google.com/search?q=SSL+Secure+Sockets+Layer+encryption)+[TLS](https://www.google.com/search?q=TLS+Transport+Layer+Security)+secure+protocol)://www.google.com/search?q=[ArcGIS]([https](https://www.google.com/search?q=HTTPS+[SSL](https://www.google.com/search?q=SSL+Secure+Sockets+Layer+encryption)+[TLS](https://www.google.com/search?q=TLS+Transport+Layer+Security)+secure+protocol)://www.google.com/search?q=ArcGIS+Esri+[GIS](https://www.google.com/search?q=GIS+Geographic+Information+System)+platform)+workspace+project+geodatabase) for [network planning](https://www.google.com/search?q=telecom+network+planning+optimization) and analysis.

## Layout

The Map view is divided into six areas:

| Area | Description |
|------|-------------|
| 1. Top bar | [Inventory3D](https://www.google.com/search?q=Cellular+Expert+Inventory3D+asset+management) window layout, settings, user menu |
| 2. Left toolbar | Data management and visualization tools |
| 3. Map canvas | Graphical data management, object placement |
| 4. Table panel | Attribute data management (bottom) |
| 5. Results panel | Data analysis tools and calculation results |
| 6. Graphs panel | Calculation result graphs, additional analysis |

## View Modes

Switch between view modes using the toolbar buttons:

- **Map view** — standard 2D map display
- **Full screen table** — expand the attribute table to full screen
- **Split vertical** — map and table side by side
- **Split horizontal** — map above, table below
